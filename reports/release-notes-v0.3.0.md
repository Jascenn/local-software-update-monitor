# Release Notes · v0.3.0

## Summary

`v0.3.0` 把这个项目从“能用的本地工具”往“可对外发布的产品版本”推进了一步。

这一版完成了：

- 对外产品名切换为 `Lingyi App Watch`
- README、CHANGELOG 和发布文档整理
- 首页交互进一步贴近可发布状态
- 保留本地优先架构，不把监控内核错误地迁移到云端

## Highlights

- `今天处理 / 风险区 / 全部资产` 三视图
- 汇总卡片可直接跳筛选结果
- 表头排序、风险优先排序和最近活动排序
- 主表全选、第三方全选和 `Shift` 连选
- 第三方审计结果与人工标记合并进同一套风险逻辑
- 网页内区分一键升级、手动处理和禁止升级

## Launch Notes

- 产品名：`Lingyi App Watch`
- 版本号：`0.3.0`
- 适合对外展示，但仍然是本地优先工具
- 官网建议放在 Cloudflare Pages
- 监控服务本体继续运行在本机
